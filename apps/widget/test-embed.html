Qivr Intake Widget Test Page

Testing the Qivr Intake Widget
This page demonstrates how the widget can be embedded in any website. The widget below is loaded from http://localhost:5173
Alternative: Direct Embed (when configured)
In production, the widget can be embedded directly into the page DOM: 
<!-- Include the widget script -->
<script src="https://your-domain.com/widget.js"></script>

<!-- Add the widget container -->
<div id="qivr-widget-root"></div>

<!-- Initialize the widget -->
<script>
  QivrWidget.init({
    tenantId: 'your-tenant-id',
    theme: {
      primaryColor: '#1976d2',
      secondaryColor: '#dc004e'
    }
  });
</script>
